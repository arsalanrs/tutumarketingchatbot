# Chat Test Questions

Here are some sample questions you can ask the Tutu Marketing Assistant to test different functionalities:

## General Data Questions

1. **What offers do we have in our Offers Library?**
   - Tests access to Offers Library data

2. **Show me all the formats available in the Formats Library**
   - Tests access to Formats Library

3. **What angles are in our Angles Library?**
   - Tests access to Angles Library

4. **What's in our Creative Matrix?**
   - Tests access to Creative Matrix data

5. **Show me the current Creative Pipeline**
   - Tests access to Creative Pipeline data

## Strategy & Analysis Questions

6. **What's the best angle for targeting first-time homebuyers?**
   - Tests AI's ability to analyze angles and provide recommendations

7. **Which format performs best for mortgage offers?**
   - Tests format analysis capabilities

8. **What offer should I use for a refinancing campaign?**
   - Tests offer recommendation logic

9. **Analyze our creative pipeline and suggest improvements**
   - Tests pipeline analysis capabilities

10. **What's the best combination of offer, angle, and format for real estate agents?**
    - Tests multi-library analysis

## Ad Creation Questions

11. **Create an ad for a 30-year fixed mortgage offer**
    - Tests ad creation functionality

12. **Create a video ad script for our refinancing offer targeting homeowners**
    - Tests format-specific ad creation

13. **Generate an ad using the "Low Interest Rates" angle and "Social Media Post" format**
    - Tests specific library item selection

14. **Create an ad for first-time homebuyers using our best performing format**
    - Tests data-driven ad creation

15. **Make an ad that combines our top offer with our most effective angle**
    - Tests intelligent combination logic

## Creative Pipeline Questions

16. **What ads are currently in our Creative Pipeline?**
    - Tests pipeline retrieval

17. **Show me ads that target real estate professionals**
    - Tests filtered pipeline queries

18. **What's the status of our latest creative ads?**
    - Tests pipeline status tracking

19. **Review the creative pipeline and tell me what's missing**
    - Tests pipeline analysis

## Advanced Questions

20. **Based on our data, what's the optimal ad strategy for Q1?**
    - Tests comprehensive data analysis

21. **Create 3 different ad variations for the same offer**
    - Tests batch ad creation

22. **What's the best angle-format combination we haven't tried yet?**
    - Tests gap analysis

23. **Compare our top 3 offers and suggest which needs more creative support**
    - Tests comparative analysis

24. **Generate a complete campaign strategy using our libraries**
    - Tests strategic planning capabilities

## Error Testing

25. **Create an ad for a product we don't have**
    - Tests error handling

26. **Show me data from a library that doesn't exist**
    - Tests invalid query handling

## Tips for Testing

- Start with simple questions to verify basic data access
- Progress to more complex queries that require analysis
- Test ad creation with different combinations
- Try edge cases and invalid inputs
- Check that the assistant properly references the libraries
- Verify that created ads are properly formatted and saved
